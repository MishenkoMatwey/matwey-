Task

Требуется реализовать операции для работы со стеком. В первой строке записано одно натуральное число N - количество операций со стеком (1<N<10000000). В каждой из следующих N строк записано по одному неотрицательному целому число. Положительное число означает, что необходимо добавить это число в стек. 0 означает, что необходимо вынуть из стека последнее число и напечатать его; если стек пуст, то ничего делать не нужно.
